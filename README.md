# link — 글방

미니멀 명함 × 미니홈피 × 작은 정원. 1페이지 SvelteKit.

**Live**: https://link-fnup.vercel.app

> 글을 쓰고 읽으며 언어를 공부합니다. 풀잎 위에 비친 하늘처럼.

## Concept

- linktr.ee의 명료함 + 싸이월드 미니홈피의 다정함
- "풀숲 · 하늘 · 연못"의 자연 쿨톤 (**Grove** 팔레트)
- 한국어 / English 언어 스위치 (예정)
- Phase 2: 익명 + 닉네임 방명록 (Supabase)

## Stack

- SvelteKit 2 + Svelte 5 + TypeScript
- Pretendard Variable (한글 가독성)
- Vercel 배포 (예정)

## Develop

```sh
npm install
npm run dev
```
