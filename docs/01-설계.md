# 컨셉

**"서재 같은 1페이지" — 미니멀 명함 × 미니홈피 × 작은 정원**

방문자가 잠깐 들러 한 호흡 쉬고 가는 곳.
linktr.ee의 명료함 + 미니홈피의 다정함 + 서재·정원의 차분함.

# 정체성 (사이트 주인)

- 글 쓰고 읽으며 **언어**를 공부함
- 관심: **인간과 자연의 조화**, **사람과 사람 사이의 깊은 연대와 평화로운 조화**

이 정체성이 톤 전체를 결정한다 — **조용함, 천천히, 깊이, 자연**.

## linktr.ee / inpock 과의 차이

| | linktr.ee | 이 사이트 |
|---|---|---|
| 본질 | 링크 리스트 (도구) | 링크 + 인상 (경험) |
| 정체성 | 사용자 무관 동일 템플릿 | 본인 고유 디자인 |
| 살아있음 | 정적 | "Now" 영역 + 방명록 |
| 떠날 때 | 클릭하고 끝 | 한 줄 남기고 갈 수 있음 |

# 디자인 시스템

## 톤
- 단일 모드 (라이트 또는 다크 하나만 — 정체성 일관)
- 액센트 1색 (또는 흑백만)
- 텍스트 그레이 4~5단계

## 타이포
- **본문**: Pretendard Variable (한글 가독성)
- **타이틀**: 가변 폰트 (variable font) — 키네틱의 핵심
  - 후보: Pretendard Variable / Migra / Fraunces Variable
- 사이즈 스케일: 1.250 (Major Third)

## 키네틱 (절제)
- 마우스 근접 시 타이틀 weight 미세 변화
- 스크롤 시 단어 단위 fade-in
- 링크 호버 시 카드 lift + 부드러운 그림자
- "Now" 갱신 시 작은 점 깜빡임

> 약속: **"살짝"** 이상 넘어가지 않는다. 데모용 효과 금지.

## 레이아웃
- 그리드 단순 1열
- 섹션 간 큰 여백이 리듬을 만듦
- 좌우 여백 충분히

```

│   ├── profile.md       # About 본문

│   └── now.md           # 현재 상태 (자주 갱신)
├── lib/
│   ├── components/

│   │   ├── About.svelte
│   │   ├── Links.svelte

│   ├── styles/

│   │   └── global.css
```

# 디자인 토큰 v1 (확정 컬러는 사용자 선택 후 채움)

## 컬러 — Grove 팔레트 (확정)

> 레퍼런스: 모네 *Water Lilies* 연작의 어우러짐 방식 + 사용자 본인의 시각 단어 "풀숲·하늘·연못".
> 그린(숲) 메인 + 블루(하늘·연못) 보조. 같은 명도 축에 머물러 부조화 없음.

```css
:root {
  /* === Grove 팔레트 — 풀숲 · 하늘 · 연못 === */

  /* 액센트 */
  --accent:        #4E6B4A;   /* 메인 — 깊은 모스 (숲) */
  --accent-deep:   #2F4A3B;   /* 가장 깊은 풀잎 (강조) */
  --accent-pond:   #4A6B7A;   /* 보조 — 연못 (블루) */
  --accent-soft:   rgba(78, 107, 74, 0.20);

  /* 배경 워시 */
  --wash-moss:  #E5EDDF;   /* 메인 풀밭 (body 그라데이션) */
  --wash-sky:   #DCE6EC;   /* 하늘 — Now 영역 */
  --wash-pond:  #E3E8EC;   /* 옅은 연못 — Guestbook 영역 (하늘 자매 톤) */

  /* 잉크 */
  --ink:        #1F2A2A;   /* 본문 — 풀그늘+물그늘 */
  --ink-muted:  #586863;
  --ink-faint:  #98A19B;
  --line:       rgba(31, 42, 36, 0.08);

  /* Surface */
  --surface:    #FBFAF6;   /* 따뜻한 오프-화이트 카드 */
}

/* Body 배경 — 풀숲(좌상) → 하늘(우하) 2축 흐름 */
body {
  background:
    radial-gradient(ellipse 95% 75% at 12% 10%, var(--wash-moss) 0%, transparent 65%),
    radial-gradient(ellipse 90% 80% at 92% 85%, var(--wash-sky) 0%, transparent 62%),
    linear-gradient(180deg, #E8EEDF 0%, #E3E9DF 45%, #DCE6EC 100%);
}
```

**컬러 사용 규칙**:
- `--accent` (깊은 모스): Hero 인용, Links 호버, Footer 강조, 메인 CTA
- `--accent-deep`: Hero `<em>` 강조, Now 메타 텍스트
- `--accent-pond`: Now meta 펄스, Guestbook 닉네임
- `--wash-sky`: Now 영역 (1개)
- `--wash-pond`: Guestbook 영역 (1개)
- 본문은 모두 `--ink` 또는 `--ink-muted`. 액센트 컬러를 본문에 쓰지 않음 (피로감)

## 타이포

```css
:root {
  --font-sans:    'Pretendard Variable', -apple-system, sans-serif;
  --font-display: 'Pretendard Variable', serif;  /* 키네틱 적용 대상 */

  /* 사이즈 스케일 (1.250 Major Third) */
  --t-xs:   0.75rem;   /* 12 */
  --t-sm:   0.875rem;  /* 14 */
  --t-base: 1rem;      /* 16 */
  --t-md:   1.25rem;   /* 20 */
  --t-lg:   1.5625rem; /* 25 */
  --t-xl:   1.953rem;  /* 31 */
  --t-2xl:  2.441rem;  /* 39 */
  --t-3xl:  3.052rem;  /* 49 — Hero */

  --leading-tight: 1.2;
  --leading-base:  1.6;
  --leading-loose: 1.8;
}
```
