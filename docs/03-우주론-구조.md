# 컨셉 — 사이트 = 하나의 우주론

방문자는 "나"라는 사람의 우주에 들어온다. 천체를 누르며 탐험.

```
        ☀ 태양 = 이성 (과학·이성·진리 탐구)
              ·
        궤도 ─────────── 🌍 지구 = 자기(自己)
                              └ 🌙 달 = 감성 (미학·인문·예술)
```

## 천체 = 정신의 구조

| 천체 | 상징 | 의미 |
|---|---|---|
| **태양** | 이성 | 과학 · 이성 · 진리 탐구 (빛·중심) |
| **달** | 감성 | 미학 · 인문 · 예술 탐구 (지구를 따라도는) |
| **지구** | 자기(自己) | 나라는 존재 = 숲 + 호수 |

## 지구를 누르면 — 지도처럼 펼쳐짐

| 영역 | 상징 | 의미 |
|---|---|---|
| **초록 숲** | 나의 밖 | 사회적 조화 (사람들 사이) |
| **푸른 호수** | 나의 안 | 내면 · 심연 (단상이 사는 곳) |

# 라우트 구조

| 경로 | 내용 |
|---|---|
| `/` | **우주 씬** — 태양·지구·달 궤도 운동, 클릭 인터랙션 (검은 배경, 동적·미니멀) |
| `/` → 지구 줌 | 지구 지도 — 숲 / 호수 클릭 |
| `/works` | **Wonnit** — 산물·포트폴리오 (Won + "we won it!") |

## 근원 → 산물

- 내면(호수)·사회(숲)·이성(태양)·감성(달)에서 오는 생각과 감정 = **근원**
- 그것이 빚어낸 결과물 = **Wonnit** (산물)

---

# 콘텐츠 배치

- **호수 (내면/심연)**: 단상 (짧은 글) ← 1차 채택
- **숲 (밖/사회)**: 사람들 사이의 조화 (추후)
- **태양 (이성)**: 과학·진리 탐구 (추후)
- **달 (감성)**: 미학·예술 (추후)
- **Wonnit**: 언어 학습 웹, 영어 학습 웹 … (외부 링크)

---

# 디자인

- 우주: 깊은 코스믹 다크 (#06070C 계열) + 옅은 별, 미세 nebula
- 태양: 따뜻한 금빛 글로우 (이성=빛)
- 지구: 작은 구 — 초록(숲) + 파랑(호수) 힌트
- 달: 옅은 라벤더-실버 (감성, 모네 라벤더의 메아리)
- 궤도: 얇은 1px 링, 낮은 불투명도
- 모션: 느리고 고요 (지구 ~42s, 달 ~9s 공전). prefers-reduced-motion 시 정지 배치
- Wonnit: 기존 Grove 라이트 테마 (타이포 중심)
